Template Installation

Once you've downloaded the archive file on TemplateMonster, extract it and locate a file called strict.zip. You can move direcory dist on hosting by using FTP client.

FTP upload

1. Using FTP client, login to the server where your website is hosted.

2. Using FTP client, move directory dist to the website root directory.

Once installation is complete, your Strict template will be ready to the settings. Download and setup the code editor like a Notepad++ and open template with editor.

Navbar

Logotype

Logo path:

header > .navbar > .container > .navbar-brand > img

You can change the logo by changing the value of the src attribute in the img tag, specifying the path to the logo in it.

Menu

Menu path:

header > .navbar > .container > .collapse > .navbar-nav > .nav-link

You can add,delete or edit menu items by copy/paste source code (for example, you can change the link by changing the value of the href attribute in the tags <a>).

Top bar

Top bar path:

header > .top-bar

You can change links to social networks as well as contact information, for this change the values of the attribute src in the tags <a>.

Also you can change the link icon by changing the value of the class attribute in the <i> tag. Icons can be taken here Font awesome.

Variations of the homepage

Template contains:

1. Default Homepage - file index.html

2. Slider Homepage - file home-variation-slider.html

3. Particles Homepage - file home-variations-particles.html

4. Video Homepage - file home-variation-youtube-video.html

Select one of these pages and rename it to the index.html, others no longer needed , you can delete their.

How to edit homepage

How to edit default homepage

Default homepage content path:

#home-section > .text-wrapper

You can change the text on the home page by searching for the <p> or <h1> tag and changing its contents.

Also you can change the button link and the text inside it by changing the value of the src attribute in the tag <a class="btn-card">.

To change the background of the home page, change the value of the style = "path" attribute of the <section> tag.

How to edit slider homepage

Slider homepage content path:

#home-section > #owl-home > .item > .container

You can change the slides by changing the contents of the <div class="item"> tags.

How to edit particles homepage

Particles homepage path:

#home-section > .particles

You can create your particles here Particles editor, then copy the config and paste it into the json constant in the config-particles.js file.

How to edit video homepage

Video homepage path:

#home-section > .player

You can change the video on the main page by changing the value of the date-attribute data-property (videoURL: '') of the tag <div class="player">.

How to edit content

The documentation will not describe the actions that are repeated, so in all text sections, the content changes in the same way. You can change the text by changing the contents of the tags p, a, span, h1, h2, h3, h4, h5, h6, button. Also, you can change the picture by changing the value of the href = "path" attribute of the img tags. To change the references, change the value of the href attribute in the a tag.

How to edit counters

You can change the counters, for this go to main.js file, find the item [1] Variable Declarations, and in it the object counters, in it you will find 4 functions, each of which is responsible for its counter. You can change the final counter digit by changing the third argument of the function to your own.

How to edit portfolio

Filter buttons path:

.portfolio-wrapper > .btn-filter-box

Portfolio path:

.portfolio-wrapper > .grid

You can add, delete or edit the elements of the portfolio. For example, you can change the name of one of the portfolio filters by changing the contents of element <a>, and all the relevant names in the main.js file in item [5] Images Loaded (Shuffle Filter). You can give the elements of the <div class="mesh-element"> portfolio the date attribute data-group = "filter_name", in order to include it in the filtering.

How to edit portfolio grid

You can add, delete or edit the elements of the portfolio grid. For example, you can add a new mesh-element element as described here. After that you can set the element <a class="action-btn"> the following date attributes data-toggle = "modal" data-target = "# modal-project1" data-image = "path" data-title = "Project title 1". This allows you to open an image in the modal window.

How to edit parallax background

You can change the parallax of the image, for this go to the main.js file and look for [4] Parallax, and then change the value of imageSrc to your own.

How to edit sidebar

Sidebar path:

#blog-list-section > .container .row .sidebar

You can add, delete or edit the contents of the sidebar, for example you can change the content of the video widget, for this, find the <div class="video widget"> and the <iframe> in it, and replace it with the one you get in youtube, as you can see here.

How to edit Google map

In order to edit the google map, you need to go to the main.js file and find there the item [14] Google Map, then change the coordinates according to the instructions that you can find here.

Footer

Footer widgets path:

footer > .container > .widget-row

You can add, delete or edit widgets in a footer. For example, to change the flickr widget you need to go here flickr badge, create a badge and copy only <div id="flickr_badge_wrapper"> and paste it into <div class="col-flickr">.
